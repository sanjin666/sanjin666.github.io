    多线程轮流打印1-100的数字

    Sanjin 4/26/2021
    • 腾讯二面面试题，在不加锁的情况下多个线程轮流打印0 - 100 的数字。
    // thread-1 : 0
// thread-2 : 1
// thread-3 : 2
// thread-4 : 3
...
// Thread-3 : 98
// Thread-4 : 99
// Thread-1 : 100

    ​ 之前学习过狂神说的JUC并发编程，在讲生产者消费者问题时，实现通过lock 的condition实现线程精准通知和唤醒线程，能做到线程轮流执行的效果，但是面试时要求不加锁就懵了，面完后思考了一会，写了一版解答。

    # 不加锁

    • 定义类变量 flag 控制线程的执行，num 实现累加
    • 采用自旋的思想， 通过 while 循环加 flag 判断实现保证线程的交替打印
    • volatile 保证变量在线程间的可见性
    • 自旋锁可以参考这篇 blog (opens new window)
    public class ThreadPrint {

    static volatile int flag = 0;
    static volatile int num = 0;

    public static void main(String[] args) {
        new Thread(()->{
            while(num <= 100){
                if(flag == 0){
                    System.out.println(Thread.currentThread().getName() + " : " + num++);
                    flag = 1;
                }
            }
        },"Thread-1").start();

        new Thread(()->{
            while(num <= 100){
                if(flag == 1){
                    System.out.println(Thread.currentThread().getName() + " : " + num++);
                    flag = 2;
                }
            }
        },"Thread-2").start();

        new Thread(()->{
            while(num <= 100){
                if(flag == 2){
                    System.out.println(Thread.currentThread().getName() + " : " + num++);
                    flag = 3;
                }
            }
        },"Thread-3").start();

        new Thread(()->{
            while(num <= 100){
                if(flag == 3){
                    System.out.println(Thread.currentThread().getName() + " : " + num++);
                    flag = 0;
                }
            }
        },"Thread-4").start();
    }
}
